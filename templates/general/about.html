Osiris is a sales agent management system that is designed to track sales agent statistics and provide performance reports based on a number of KPI's. The system uses daily stat and weekly stat input to generate a series of reports that are viewable by the agents and supervisors

Designed and Built by David Sivocha using the Django Framework

Includes the following modules:

  • Jquery
  • The Orbit slider from Zurb Foundation
  • The wonderful charts interface from Highcharts
  • The Tablesorter Jquery plugin
  • The Masked Input Jquery plugin
  • Django-Admin-Tools extension for Django
  • Django-Admin-Tools-Bootstrap modification of Django-Admin-Tools

Built using Bootstrap from Twitter
